# Images

Drop screenshots, diagrams, and OG card art here.

The site looks for these filenames automatically and will swap them in when present:

- `shot-01.jpg` — Staged gallery, recognition active
- `shot-02.jpg` — Anchored clue overlay
- `shot-03.jpg` — Living portrait moment
- `shot-04.jpg` — Themed-hunt selection UI
- `shot-05.jpg` — Hunt completion state
- `shot-06.jpg` — Generation pipeline output
- `demo-poster.jpg` — Poster frame for demo video
- `og-card.png` — 1200×630 social share image

If a file is missing, the typographic placeholder remains visible — no broken images.
