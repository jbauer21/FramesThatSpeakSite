# Videos

Drop the demo video here as `demo.mp4` and (optional) captions as `demo.vtt`.

To swap the placeholder for a real `<video>` element, open `index.html`,
find the `<figure class="video-wrap">` block, delete the
`<div class="video-placeholder">…</div>`, and uncomment the `<video>` tag
already provided directly below it.

Recommended:
- 1080p, H.264, ≤ 100 MB for GitHub Pages friendliness.
- Captions in `demo.vtt` (WebVTT). Even rough captions help WCAG 1.2.2 conformance.
